Como Funciona? [Mostrar]

Este sistema automatiza as regras de encontros aleatórios de Tormenta20. A cada dia de viagem (ou cena) sem encontros, a chance de perigo aumenta em 5%.

1. Selecione o Patamar do grupo (+30/+70/+110 na rolagem) e o Terreno para gerar os monstros.
2. Use o Diário de Viagem para gerenciar os dias e rolar encontros automáticos.
3. Se um encontro for detectado, o gerador de desafios se abrirá para determinar os detalhes.
4. Criaturas encontradas serão associadas ao bestiário. Você poderá ver a ficha e enviá-las diretamente para o combate!
